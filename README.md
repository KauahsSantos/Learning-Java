# 📚 Estudos: Desenvolvimento com Java  

Este roadmap foi criado para guiar o aprendizado completo da linguagem **Java**, desde os fundamentos até aplicações avançadas no mercado.

## 📋 Estrutura do Roadmap  

- [ ] **Introdução ao Java**  
  - [ ] O que é Java?  
  - [ ] História e filosofia da linguagem  
  - [ ] Instalação do JDK  
  - [ ] Configuração do ambiente (VS Code / IntelliJ IDEA)  
  - [ ] Primeiro programa (`Hello World`)  
  - [ ] Estrutura básica de um programa Java  

- [ ] **Fundamentos da Linguagem**  
  - [ ] Variáveis e tipos de dados  
    - [ ] Tipos primitivos (int, double, char, boolean)  
    - [ ] Tipos de referência  
  - [ ] Operadores  
    - [ ] Aritméticos  
    - [ ] Lógicos  
    - [ ] Relacionais  
  - [ ] Entrada e saída de dados (`Scanner`, `System.out`)  
  - [ ] Controle de fluxo  
    - [ ] if / else  
    - [ ] switch  
    - [ ] loops (for, while, do-while)  

- [ ] **Métodos e Organização de Código**  
  - [ ] O que são métodos  
  - [ ] Parâmetros e retorno  
  - [ ] Sobrecarga de métodos (Overloading)  
  - [ ] Escopo de variáveis  
  - [ ] Boas práticas de organização  

- [ ] **Programação Orientada a Objetos (POO)**  
  - [ ] Conceitos fundamentais  
    - [ ] Classe e objeto  
    - [ ] Atributos e métodos  
  - [ ] Encapsulamento  
  - [ ] Herança  
  - [ ] Polimorfismo  
  - [ ] Abstração  
  - [ ] Modificadores de acesso (public, private, protected)  
  - [ ] Construtores  

- [ ] **Coleções e Estruturas de Dados**  
  - [ ] Arrays  
  - [ ] ArrayList  
  - [ ] LinkedList  
  - [ ] HashMap  
  - [ ] HashSet  
  - [ ] Iteração com loops e iteradores  
  - [ ] Generics  

- [ ] **Tratamento de Exceções**  
  - [ ] O que são exceções  
  - [ ] try, catch, finally  
  - [ ] throw e throws  
  - [ ] Criação de exceções personalizadas  

- [ ] **Manipulação de Arquivos (I/O)**  
  - [ ] Leitura de arquivos  
  - [ ] Escrita de arquivos  
  - [ ] Streams  
  - [ ] Serialização  

- [ ] **Programação Funcional (Java Moderno)**  
  - [ ] Expressões Lambda  
  - [ ] Streams API  
  - [ ] Optional  
  - [ ] Programação declarativa  

- [ ] **Banco de Dados com Java**  
  - [ ] Introdução ao JDBC  
  - [ ] Conexão com banco de dados  
  - [ ] CRUD (Create, Read, Update, Delete)  
  - [ ] Integração com MySQL/PostgreSQL  

- [ ] **Desenvolvimento Web com Java**  
  - [ ] Introdução ao backend  
  - [ ] Servlets  
  - [ ] JSP (Java Server Pages)  
  - [ ] Introdução ao Spring Framework  
    - [ ] Spring Boot  
    - [ ] APIs REST  
    - [ ] Injeção de dependência  

- [ ] **Testes e Qualidade de Código**  
  - [ ] Testes unitários com JUnit  
  - [ ] Testes de integração  
  - [ ] Boas práticas (Clean Code)  

- [ ] **Ferramentas e Build**  
  - [ ] Maven  
  - [ ] Gradle  
  - [ ] Gerenciamento de dependências  

- [ ] **Concorrência e Multithreading**  
  - [ ] Threads  
  - [ ] Runnable e Callable  
  - [ ] Sincronização  
  - [ ] Executors  

- [ ] **Desenvolvimento Mobile (Android Nativo)**  
  - [ ] Introdução ao Android com Java  
  - [ ] Estrutura de um app Android  
  - [ ] Activities e Layouts  
  - [ ] Ciclo de vida de Activities  
  - [ ] Integração com APIs  

- [ ] **Práticas Avançadas**  
  - [ ] Design Patterns (Singleton, Factory, etc.)  
  - [ ] Arquitetura de software  
  - [ ] Clean Architecture  
  - [ ] SOLID  

- [ ] **Projetos Práticos**  
  - [ ] Sistema de cadastro (CRUD no terminal)  
  - [ ] Sistema bancário simples  
  - [ ] API REST com Spring Boot  
  - [ ] Sistema com banco de dados  
  - [ ] Aplicação Android simples  

## 📚 Recursos Recomendados  

- [ ] Documentação oficial do Java (Oracle)  
- [ ] Cursos na Udemy, Alura ou Rocketseat  
- [ ] Prática no HackerRank / LeetCode  
- [ ] Comunidade no Stack Overflow  

## 🎯 Meta Final  

Dominar Java ao ponto de conseguir:

- Desenvolver aplicações completas (backend ou desktop)  
- Criar APIs robustas com Spring Boot  
- Trabalhar com banco de dados  
- Construir projetos reais para portfólio
